# Donnees d'exemple

Ce repertoire contient des documents de cours utilises pour tester le pipeline d'ingestion RAG.
Le fichier `lecture_ml.md` est un support de cours en francais sur l'introduction au Machine
Learning supervise ; il couvre les definitions, les principaux algorithmes (regression lineaire,
arbre de decision, SVM, kNN), les metriques d'evaluation et la problematique du surapprentissage.
Ces fichiers ne sont pas destines a la production et peuvent etre supprimes ou remplaces par de
vrais supports de cours sans impacter le fonctionnement de l'application.
